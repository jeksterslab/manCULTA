# Simulation Results (Entropy)

Simulation Results (Entropy)

## Usage

``` r
data(results_entropy)
```

## Format

A dataframe with 14 rows and 4 columns:

- TaskID:

  Simulation Task ID.

- N:

  Sample size.

- Model:

  Fitted model.

- Entropy:

  Mean entropy.

# Write Data to File

Generic function to write data to file.

## Usage

``` r
WriteData(x, file, ...)

# S3 method for class 'simculta'
WriteData(x, file, ...)
```

## Arguments

- x:

  Object of class `simculta`.

- file:

  Character string. File name.

- ...:

  Additional arguments.

## Value

Invisibly returns `NULL`. Writes data to file as a side effect.

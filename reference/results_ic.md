# Simulation Results (Information Criteria)

Simulation Results (Information Criteria)

## Usage

``` r
data(results_ic)
```

## Format

A dataframe with 60 rows and 5 columns:

- TaskID:

  Simulation Task ID.

- N:

  Sample size.

- Model:

  Fitted model.

- IC:

  Information criteria type.

- Value:

  Mean information criteria.
